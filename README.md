# A Tour of Deep Learning

An overview of the core concepts of deep learning, with pointers to further reading.

## Sources

The content and structure draw heavily on two primary sources:

- **Understanding Deep Learning** by Simon J.D. Prince (freely available at [udlbook.github.io/udlbook](https://udlbook.github.io/udlbook/))
- **Deep Learning Lecture, J.G. University Mainz** by Andreas Hildebrandt

Additional references are cited throughout the slides.
